My interest for tech started from a young age when I successfully installed a Pink Panther game as a boy despite my father's failure at it. Since then, I have loved playing and tinkering with software, whether it would be smaller scripts or modifying games. In the summer of 2023 I finally made the jump from being a financial analyst to pursue a lifelong dream/challenge as a software developer.

Whether it would be scripting or using cheat engine to bend the games towards my will, or hosting Minecraft servers for me and my friends and others to join by adventurously opening the ports of my home network, the computer has been a large part of my life. However, my first real step into coding would first start after I completed a B.A in Accounting and Finance and worked in that industry for a couple of years. That experience pushed me into finally taking my hobby as a serious career. It all started with a Java Bootcamp which did not go as I expected, which led me to pursue courses online, start a interest in game development and finally start attending Chas Academy as a System Developer using C/C++.
